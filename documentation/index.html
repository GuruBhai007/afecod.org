Common WordPress Installation Issues

Missing style sheet error when installing the theme

A common issue that can occur with users new to installing WordPress themes is a “Broken theme and/or stylesheets missing” error message being displayed when trying to upload or activate the theme. This error message does not mean that the theme you have purchased is broken, it simply means it has been uploaded incorrectly. Luckily, there is a very easy fix.

 

How to get started

How to get your website like our demo!

Before you actually get into the import process, please make sure the supported plugins for The Company are installed.

You can import the dummy XML file (found in the download package) to get your website like our demo. You can import the XML file via "Tools -> Import" in the WordPress admin area. Setting up the navigation menu

There are three navigation menu locations in this theme. By default they aren't used and can cause the theme not to look the way you want. You need to set them up under "Appearance -> Menus".

The menu locations that are available are:

  • Main menu: This is the top navigation menu, and the main menu for your users to navigate through your website.
  • Footer menu: This is the footer menu, and located at the very bottom of your website. This menu hides automatically on smaller screens.
  • Mobile main menu: This will be the top navigation menu on mobile devices, and will replace the normal main menu. This way you can slim down the menu, or present users with other pages on their mobile phone.

 

Setting up the frontpage

The frontpage that we've set up has two important parts that need to be set up: A section page and a static frontpage.

Custom frontpage

If you want to add content to your frontpage then you´ll need to set this up in the WordPress settings. By default WordPress displays a list of your blog posts. To set up a static frontpage, go to "Settings -> Reading". Select the page that you'd like to put on the frontpage. Please note that the frontpage won't display the page title since that's not something you'd want for the frontpage. The rest is exactly the same as on other pages. For the demo we've set up the frontpage with a section page which we'll explain in the next section: Creating a section page.

 

 

Creating a sections page

The sections page can be used for a lot of things. You can use it to build a page with many different rows of content, but you can also use it to re-use content on different pages.

Creating a sections page requires two important steps:

Creating sections

First of all you'll need to create sections. To do this, follow these steps:

  1. Go to "Sections -> Add New".
  2. Insert a title for this section. This title will not be shown on the website and is purely to keep them apart in the backend.
  3. Select the section type in the section options. The 'Plain content' option will display the section with Standard content, the 'Portfolio' option will allow you to select portfolio items to show in this section, the 'Widget area' option will give you the option to display one of the sidebars you've created and the 'Blog tiles' option will display blog posts in the same layout the portfolio is dispayed.
  4. If you've selected the 'Plain content' option, you can select a Featured Image for your section. This image will automatically cover the whole background of the section.
  5. Each section type has its own set of options. However, there are several options that all sections share: The section background and text color, and the option to specify the top and bottom margin. You can also give the section a negative margin to move the content up and down. If you give it enough negative margin you can even make it hover over another section. This is especially nice with images.

The section page will display all the sections that you selected yourself. To do this, follow these steps:

  1. Go to "Pages -> Add New".
  2. Insert a title for your page.
  3. For now skip the content and select the "Sections page" page template on the right hand side of the page.
  4. In the "Section Page Options" you can select which sections to display. We've chosen to list all the created sections here so you can select which ones you want to display. You can also reorder the sections by dragging and dropping them.
  5. You can create several section pages like this with different (or the same) sections.

Creating a portfolio

The portfolio can be used to display your products, services or work you've done. Once the portfolio items have been created you can display them via a portfolio section. This also means you can combine the portfolio with a lot of other content.

Creating a portfolio requires two important steps:

Creating portfolio items

First of all you'll need to create portfolio items. To do this, follow these steps:

  1. Go to "Portfolio items -> Add New Item".
  2. Insert a title for this portfolio item. This is shown in the portfolio overview and on the portfolio item page itself. Make sure it's catchy and not to long.
  3. Insert the content that you want. You can use columns to devide text and images for a better look. You're completely free in this.
  4. Add your Featured Image. You can do this on the right side of the page. Be sure to do this because this is a required step! Any size will do since the image will automatically cover the tile in the portfolio overview.
  5. Additionally you can also use sections in the portfolio. This way you can reuse call to action areas or set up advanced portfolio items.

Creating a portfolio section

Once you've created some portfolio items you can start setting up a portfolio section to display these portfolio items. This is already explained in the Creating a sections page chapter, but for this example we'll zoom in on the portfolio type section. To set up your portfolio section, follow these steps:

  1. Go to "Sections -> Add New".
  2. Insert a title for this section. This title will not be shown on the website and is purely to keep them apart in the backend.
  3. Select the "Portfolio" section type in the Additional settings. This will reveal the "Portfolio Section Options".
  4. Now you can set up some colors for your portfolio section. You can also set how many columns you'd like to display.
  5. More important, you can set up your portfolio filter and which portfolio items you want to display. For flexibility reasons we haven't coupled these settings. You can select any category and any portfolio item you want. Feel free to drag and drop the items to reorder them.
  6. That's it. Save the page and you've created your very own portfolio section. Now you can load it on any section page you want. Your frontpage, a portfolio page or maybe on a services page? The choice is yours.

Using bootstrap elements

This theme uses Twitter Bootstrap as a strong foundation. This also means that there are a ton of integrated features that you can make use of out of the box. To do this easily you can install the “Easy Bootstrap Shortcode” plugin. You can download the plugin here: http://wordpress.org/plugins/easy-bootstrap-shortcodes/

This plugin adds buttons to your content area for easy adding the Twitter Bootstrap elements to your page. We've made an example page of all the shortcodes here: http://www.redfactory.nl/themes/thecompany/shortcodes/

Alternatively you can also use the included Visual Composer plugin. This also uses Twitter Bootstrap as a foundation, but allows you to set up all your content in an easy and visual way. We highly recommend using this plugin!

Translating this theme

The theme can be completely translated. There are two ways to do this. You can either use pot/po/mo files or you can use a plugin.

Using a plugin

You can use a plugin to translate all the text in a theme. The "WPML" plugin, for example, is perfect for this, and will allow you to set up several languages: http://wpml.org/

Using pot/po/mo files

You can also translate the theme using pot/po/mo files. To get the files that you need you first need to get all the strings from the theme. You can do this with a plugin like "Loco Translate": http://wordpress.org/plugins/loco-translate/.

This plugin will allow you to create and load the files that you need. There are also other plugins for this ofcourse, so use the one that you like best!

Customizations

Making modifications to a theme can be a tricky business. What if there's an update to a theme, do you need to overwrite all the changes you've made? WordPress has the perfect solution right out of the box: Child Themes.

A Child Theme can be seen as a completely seperate theme, but it loads all the files from its parent. Only the files that you modify will be overwritten. A complete tutorial can be read on the WordPress website: http://codex.wordpress.org/Child_Themes

If you don't want to do these customizations yourself you can let skilled developers take on the task for you! Post your theme customizations on Theme Customizers: http://www.themecustomizers.com/
